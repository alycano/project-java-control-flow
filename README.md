 Project Java Control Flow

Este proyecto corresponde a la actividad GA1-220501096-04-AA1-EV02: Construcción de algoritmos en Java utilizando estructuras de control condicional y bucles, del programa Tecnología en Análisis y Desarrollo de Software** del SENA.

Su propósito es implementar y comprender las principales estructuras de control de flujo en Java (`if`, `if-else`, `switch`, `for`, `while`, `do-while`), aplicadas a problemas cotidianos.  
El avance se documenta mediante commits progresivos en Git y GitHub.
